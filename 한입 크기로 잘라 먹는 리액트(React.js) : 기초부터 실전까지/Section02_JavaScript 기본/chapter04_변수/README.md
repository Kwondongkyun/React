# 변수

### 1. 변수
```javascript
let age;
age = 30;
```

### 2. 상수
```javascript
const birth = "2001.01.01";
```

### 3. 변수 명명규칙(네이밍 규칙)
- $, _ 제외한 기호는 사용할 수 없다.
  ```javascript
  let $_name;
  ```

- 숫자로 시작할 수 없다.
  ```javascript
  let name1;
  let _2name;
  ```

- 예약어를 사용할 수 없다.
  ```javascript
  let if;  // 오류
  let name;
  ```
